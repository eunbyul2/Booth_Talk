```javascript

// 출력: "10.29(수) 11:00"

- 설명
- 혜택 정보

- 이벤트명, 업체명, 설명, 부스번호로 검색 가능

## 🔄 데이터 흐름

### 현재 (MOCK 데이터)
```

```

```

## 🔧 향후 작업

### Backend API 연동 시
```javascript
// API 호출 예시
const fetchExhibitionEvents = async (exhibitionId) => {
  const response = await fetch(`/api/visitor/events/exhibition/${exhibitionId}`)
  const data = await response.json()
  return data
}

// 사용
useEffect(() => {
  fetchExhibitionEvents(1).then(data => {
    setExhibition(data.exhibition)
    setEvents(data.events)
  })
}, [])
```
